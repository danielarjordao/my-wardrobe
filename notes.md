# Notes

- **ngModel**: Esse é um recurso do Angular que permite a ligação bidirecional entre o modelo de dados e a interface do usuário. No código fornecido, o `[(ngModel)]="searchTerm"` está vinculando o valor do campo de entrada ao atributo `searchTerm` no componente TypeScript. Isso significa que qualquer alteração feita no campo de entrada atualizará automaticamente o valor de `searchTerm` no componente, e vice-versa.
